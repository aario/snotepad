Add a Note Folder

You can add existing folders from your phone's storage to SNotePad to manage the notes within them.

  1. Navigate to the Settings page (usually via a gear icon in the sidebar).
  2. Find the "Folders" section.
  3. Tap the Plus button.
  4. Your phone's file browser will open. Navigate to and select the folder you wish to add.
  5. Confirm the selection (the exact step depends on your Android version).
  6. The selected folder will now appear in the Settings list and the sidebar.

Remove a Note Folder

Removing a folder from SNotePad only removes it from the app's list; it does not delete the folder or its contents from your phone.

  1. Navigate to the Settings page.
  2. Find the folder you want to remove in the "Folders" list.
  3. Tap the Trash Can icon next to the folder name.
  4. Confirm the action in the dialog box that appears.
  5. The folder will be removed from the Settings list and the sidebar.

Reorder Note Folders

You can change the order in which folders appear in the sidebar.

  1. Navigate to the Settings page.
  2. In the "Folders" section, tap and hold the folder you want to move.
  3. Drag the folder up or down to the desired position in the list.
  4. Release your finger. The new order is saved automatically.

Change the App Theme

Customize the appearance of SNotePad.

  1. Navigate to the Settings page.
  2. Find the "Theme" section.
  3. Select your preferred theme:
    • Light : Standard light background.
    • Dark : Dark background, light text.
    • Auto : Automatically switches between Light and Dark based on your system settings.
  4. The theme change is applied immediately.

Navigate Between Folders

Quickly switch between your added note folders.

  1. Open the Sidebar (if it's hidden, tap the menu icon in the top bar).
  2. Under the "Folders" heading, tap the name of the folder you want to view.
  3. The main view will update to show the contents of the selected folder.

Viewing Notes in a Folder

When you select a folder, its contents (note files) are displayed.

  • Each note is shown in a card format.
  • You'll see the filename and the last modified date.
  • Tap anywhere on the note's card (except the delete button) to open it in the editor.

Sort Notes in a Folder

Change the order in which notes are displayed within a folder.

  1. While viewing a folder's contents, locate the Sort icon (or similar) in the top navigation bar.
  2. Tap the Sort icon to open the sorting options dropdown.
  3. Select your desired sorting method:
    • Name A - Z
    • Name Z - A
    • Date Oldest First
    • Date Newest First
  4. The file list will automatically reorder based on your selection.

Search for Notes in a Folder

Quickly find notes by searching their names or content within the current folder.

  1. While viewing a folder's contents, locate the Search bar in the top navigation bar.
  2. Tap into the search bar and type your search query.
  3. The list of notes will automatically filter as you type to show only matching files. The search looks through filenames and the content of the notes.
  4. Clear the search bar to see all notes in the folder again.

Create a New Note

  1. Navigate to the folder where you want to create the new note using the sidebar.
  2. Tap the Plus button located in the header of the folder view.
  3. The editor screen will open with a blank note.
  4. Tap into the filename input field at the top and type a name for your new note (e.g., "My New Idea").
  5. Start typing your note content in the main editor area.
  6. Tap the Save icon to save the new note file.

Edit a Note (Using Markdown)

SNotePad uses Markdown for formatting. It's a simple way to add structure and style.

  1. Open the note you want to edit by tapping on it in the folder view.
  2. The note content will appear in the editor.
  3. Tap into the editor area and make your changes.
  4. Use Markdown syntax or the toolbar buttons for formatting:
    • **Bold text** or __Bold text__ for Bold
    • *Italic text* or _Italic text_ for Italic
    • # Heading 1, ## Heading 2, etc. for Headings
    • * Item 1 or - Item 1 for unordered lists
    • 1. Item 1 for ordered lists
    • > Quoted text for blockquotes
    • `Inline code` for inline code
    • ```
      Code block
      ``` for code blocks
    • [Link text](https://example.com) for links
    • ![Alt text](https://example.com/image.jpg) for images
    • ~~Strikethrough~~ for Strikethrough
    • --- or *** for horizontal rules
  5. Remember to save your changes using the Save icon .

Search Within a Note

Find specific text inside the note you are currently editing.

  1. While editing a note, locate the Search bar in the top navigation bar (the one above the editor, not the main app bar).
  2. Tap into the search bar and type the text you want to find.
  3. As you type, all occurrences of the text within the note will be highlighted.
  4. The editor might scroll to the first match automatically.
  5. Clear the search bar to remove the highlights.

Save Your Note

Save the changes you've made to a note.

  1. While editing a note, tap the Save icon in the top navigation bar.
  2. A confirmation toast message will appear indicating the note has been saved.
  3. If it's a new note, it will be saved with the filename you entered at the top. If you renamed an existing note, it will be saved with the new name in the same folder.

See also: Understand Autosave

Rename a Note

Change the filename of an existing note.

  1. Open the note you want to rename in the editor.
  2. Tap into the filename input field at the very top of the editor screen (above the toolbar).
  3. Edit the filename as desired. Make sure it's a valid filename for your system.
  4. Tap the Save icon.
  5. The note will be saved with the new filename in its original folder.

Delete a Note

Permanently remove a note file from your device.

  1. Navigate to the folder containing the note you want to delete.
  2. Find the note's card in the file list.
  3. Tap the Trash Can icon on the right side of the note's card.
  4. A confirmation dialog will appear. Tap Confirm (or similar) to permanently delete the file.
Warning: Deleting a note is permanent and cannot be undone through the app.

Insert an Image

Add images to your notes.

  1. Place the cursor in the editor where you want the image to appear.
  2. Tap the Image icon in the editor toolbar.
  3. If prompted, enter the URL (web address) of the image and tap OK. The Markdown syntax for the image will be inserted.
  4. Alternatively, if image uploads are configured:
    • Tap the specific Upload Image icon (if available).
    • Select an image file from your device.
    • The image will be uploaded (if applicable) and the Markdown syntax pointing to it will be inserted automatically.
    • You might also be able to drag and drop an image file directly into the editor area or paste an image from your clipboard.
  5. Edit the placeholder alt text (inside the square brackets `[]`) to describe the image.

Use the Formatting Toolbar

Quickly apply Markdown formatting using the toolbar above the editor.

  • Select Text & Click: For most formatting (like Bold , Italic , Strikethrough ), select the text you want to format first, then click the corresponding toolbar button.
  • Click & Type: For some elements (like Headings , Lists , Quotes , Code ), place your cursor on a new line or at the start of a line and click the button. The necessary Markdown syntax will be inserted, and you can start typing.
  • Links & Images : Clicking these buttons will usually insert the basic Markdown structure (like `[]()` or `![]()`) and may prompt you for the URL and optional title/alt text.
  • Clean Block : Select a block of text and click this to remove most Markdown formatting characters from it.
  • Horizontal Rule : Inserts a horizontal rule (`---`) at the cursor position.
  • Table : Inserts a template for a Markdown table at the cursor position.

Preview Your Formatted Note

See how your Markdown note will look when rendered.

  1. While editing, tap the Preview icon in the editor toolbar.
  2. The editor view will be replaced by a read-only preview of the formatted note.
  3. Tap the Preview icon again to return to the editing mode.

Use Side-by-Side Editing and Preview

View your Markdown source and the live preview simultaneously (best on larger screens).

  1. Tap the Side-by-Side icon in the editor toolbar.
  2. The screen will split, showing the editor on one side and the live preview on the other.
  3. As you type in the editor, the preview will update automatically.
  4. Tap the Side-by-Side icon again to return to the normal editor view.

Use Fullscreen Mode

Maximize the editing or preview area for a distraction-free experience.

  1. Tap the Fullscreen icon in the editor toolbar.
  2. The editor (or preview/side-by-side view if active) will expand to fill the entire screen.
  3. Tap the Fullscreen icon again to exit fullscreen mode.

Understand Autosave

SNotePad can automatically save your work periodically.

  • If enabled in the settings (usually requires a unique ID for the app instance), the app will automatically save the current note to local storage a short while after you stop typing.
  • A status message (often in the bottom status bar) might indicate when the last autosave occurred (e.g., "Autosaved: 10:00 AM").
  • This provides a safety net against accidental data loss but does not replace manually saving the file using the Save icon, which writes it permanently to your phone's storage.
  • If the app closes unexpectedly and you reopen it to the same note, it may load the autosaved version.

Undo or Redo Changes

Easily revert or reapply actions in the editor.

  • Tap the Undo icon in the toolbar to reverse your last action(s).
  • Tap the Redo icon in the toolbar to reapply action(s) you just undid.

Understanding How Notes Are Saved

A key feature of SNotePad is that it saves your notes as simple text files directly in the folders you choose on your phone. This is different from many apps that store notes inside a private database.

Benefits of this approach:

  • Full Control: Your notes are just files. You can browse them using your phone's file manager app.
  • Easy Backup: You can back up your notes simply by copying the files or the entire folder using standard backup methods (like copying to a computer, cloud storage, or SD card).
  • Portability: Since they are standard files (likely plain text or Markdown `.md`), you can open and edit them with many other text editors on any device.
  • No Lock-in: You're not tied to SNotePad. If you decide to switch apps, your notes are already in an accessible format.
  • Simple Sharing: Share individual note files easily using your phone's standard sharing options.
